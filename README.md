#### Мой проект KeyDivide 

В этом репозитории представлен код сайта по продаже эргономичных клавиатур. Фронтенд написан на базовом стеке - HTML, CSS, JS. Для бэкенда выбрана базы данных PostgreSQL, а для работы с ней используются Node.js + Express.js

Прошивка для клавиатур, представленных на сайте, находится в следующем репозитории - [Split_Keyboard](https://github.com/Neyman39/Split_Keyboard)
